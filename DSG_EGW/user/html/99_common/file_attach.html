파일첨부 공통양식

각각에 대해 tr부분부터 구조 동일하게 사용하시면 됩니다.
기본형은 "닫힘 상태 + 가이드 텍스트" 가 기본이며, 파일첨부 스트링을 a 태그 내에 배치하여 클릭시 영역을 오픈하도록 수정되었습니다.
다만, 첨부버튼 클릭시-에도 해당 영역이 펼쳐지도록 부탁드립니다.
p.s. 상세 동작 및 ui 가이드는 기획서 '~주소찾기.첨부파일.조직도팝업_v0.7~' 6페이지 이하 참고 부탁드립니다.
